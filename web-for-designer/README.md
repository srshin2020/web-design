## VS extension
Auto Close Tag - 자동으로 태그를 닫아주는 기능
Auto Rename Tag - 시작태그를 수정하면 종료태그를 같이 바꿔주는 기능
HTML Snippets - 효율적인 태그 작성을 도와주는 기능
Autoprefixer - 크로스 브라우징을 위해 접두사를 넣어주는 기능
Live Server - 로컬서버를 만들어 브라우저에서 실행되고 저장하면 바로 적용을 보여주는 기능
